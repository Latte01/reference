点击知识库首页上的“**新建知识库**”按钮，打开如下新建知识库的页面。

<img
    src="docs/images/knowledge-new.png"
    alt="new knowledge"
/>

根据表单中的字段进行填写，其中“可见性”仅管理员组的成员可使用。

可见性说明：

| 可见性          | 说明                                                      |
| -------------- | -------------------------------------------------------- |
| 私有            | 知识库仅管理员组内可见，其他组不可见                           |
| 完全公开        | 公开知识库，平台内均可见                                      |
| 仅详情页可见     | 对于管理员组以外的其他组，仅可见知识库的详情页，知识库内的文件不可见 |
