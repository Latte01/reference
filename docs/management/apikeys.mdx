平台使用 API Key 进行身份验证。访问您的 API Keys 页面，获取您在请求中使用的 API Key。API Keys的管理页面如下图所示。

<Tip>为了保护您的账户安全，请勿与他人分享您的 API Key，或在浏览器或其他客户端代码中公开您的 API Key。</Tip>

## 新建 API Key
1. 在左侧导航栏中选择“个人设置”>“API Keys”。
2. 在 API Keys 列表页中，单击“新建Key”来创建一个API Key，打开“新建API Key”窗口。

<img
    style={{height: "250px", border: "1px solid #EEE"}}
    className="block dark:hidden"
    src="docs/images/api-key-new.png"
    alt="apikey"
/>

3. 根据页面提示填写相关信息，可以在创建 API Key 的过程中将Key进行保存，后续将不可见。
4. 单击“确定”，完成 API Key 的创建。

## 编辑 API Key
点击 API Key 对应操作列中的“编辑”按钮，可以对该 API Key 进行编辑。编辑 API Key 只能修改其名称，不可修改其Key。


## 删除 API Key
点击 API Key 对应操作列中的“删除”按钮，可以删除该 API Key，API Key 删除后不可恢复。为了防止 API Key 泄露而造成的账号风险，您可以将该 API Key 删除，创建一个新的 API Key 进行使用。
