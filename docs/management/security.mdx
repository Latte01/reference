平台支持用户对个人账号进行管理，个人账户的安全设置主要用于保护用户的个人信息和账户安全，防止个人信息被滥用或盗窃。通过安全设置，用户可以采取一系列措施来增强账户的安全性，例如修改手机号码、电子邮箱、登录密码等。

<Tip>为了保护您的账户安全，请通过加强密码强度，定期修改密码等方式进行保护。</Tip>

## 手机号码

平台可以使用手机号码作为用户的登录账号，用户可以根据实际需要修改手机号码，以保证其账号的安全性。

在编辑手机号码的过程中，为了降低账号风险，流程中加入了账号验证机制，以防止手机号码被随意修改。

编辑流程如下：
1. 通过点击“手机号码”一栏中的“编辑”按钮，进入编辑手机号码的流程。
2. 身份验证，要求用户输入当前账号的密码以验证其身份。

<img
    style={{width: "320px", border: "1px solid #EEE"}}
    className="block dark:hidden"
    src="docs/images/security-phone-check.png"
    alt="check"
/>

3. 输入新手机号码，及其验证码，以验证该手机号码的有效性及用户身份。

<img
    style={{width: "320px", border: "1px solid #EEE"}}
    className="block dark:hidden"
    src="docs/images/security-phone-new.png"
    alt="phone"
/>

4. 点击“确定”，完成用户手机号码的编辑。

## 电子邮箱

平台可以使用电子邮箱作为用户的登录账号，用户可以根据实际需要修改电子邮箱，以保证其账号的安全性。

在编辑电子邮箱的过程中，为了降低账号风险，流程中加入了账号验证机制，以防止电子邮箱被随意修改。

编辑流程如下：
1. 通过点击“电子邮箱”一栏中的“编辑”按钮，进入编辑电子邮箱的流程。
2. 身份验证，要求用户输入当前账号的密码以验证其身份。

<img
    style={{width: "320px", border: "1px solid #EEE"}}

3. 输入新电子邮箱，点击“获取验证码”，系统将向该电子邮箱地址发送验证码，用户输入验证码以验证该电子邮箱的有效性及用户身份。

<img
    style={{width: "320px", border: "1px solid #EEE"}}
    className="block dark:hidden"
    src="docs/images/security-email-new.png"
    alt="email"
/>

4. 点击“确定”，完成用户电子邮箱的编辑。

## 登录密码

定期修改账号登录密码，以保证其账号的安全性。

在修改登录密码的过程中，为了降低账号风险，流程中加入了账号验证机制，以防止用户的账号密码被随意修改。

编辑流程如下：
1. 通过点击“登录密码”一栏中的“编辑”按钮，进入编辑登录密码的流程。
2. 身份验证，要求用户输入当前账号的密码以验证其身份。

<img
    style={{width: "320px", border: "1px solid #EEE"}}
    src="docs/images/security-password-check.png"
    alt="check"
/>

3. 设置新密码，输入新密码及新密码的确认密码。

<img
    style={{width: "320px", border: "1px solid #EEE"}}
    src="docs/images/security-password-new.png"
    alt="password"
/>

4. 点击“确定”，完成用户账号密码的编辑。

## 退出组织

退出组织，指用户退出当前组织。退出当前组织后用户将服务使用该组织内的资源，需要组织管理员邀请后才能加入该组织。
