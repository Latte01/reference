在专属定制页面点击 `专属定制` 按钮来定制专属大模型。

用户需要根据表单要求填写和配置定制专属大模型所需要的内容，其设置步骤及每部分的说明如下。

## 基本信息

`基本信息` 部分如下图所示，用户根据实际情况进行填写。

<img
    src="docs/images/llm-new-basic.png"
    alt="new LLM"
/>

字段说明：
| 字段         | 说明                          |
| ----------  | ----------------------------- |
| 名称        | 必填，大模型的唯一标识，保存后不可更改，请根据页面上的要求设置其名称  |
| 显示名称     | 必填，显示在卡片和详情页上的大模型名称，保存后可编辑               |
| 标签        | 非必填，给大模型设置标签                                       |
| 头像        | 非必填，给大模型设置一个头像                                    |
| 描述        | 非必填，`Markdown` 格式，其内容将显示在大模型详情页的 `描述` 中    |

## 个性化设定

`个性化设定` 部分如下图所示，给专属大模型设置个性化能力，并根据您的喜好进行对话。

<img
    src="docs/images/llm-new-individual.png"
    alt="new LLM"
/>

字段说明：
| 字段         | 说明                              |
| ----------  | -------------------------------- |
| 角色        | 非必填，给大模型设置一个角色          |
| 人个设定     | 非必填，通过一段描述来设定大模型的人格  |
| 创造性       | 必填，`0-2之间` 的参数，数值越大其创造性越强，其生成响应时更加注重创造性和独特性。|
| 情绪        | 必填，给大模型设置一个情绪，可选有 `友好`，`正式`，`轻松`，`兴奋`                                    |
| 通识模型     | 必填，选择一个系统中预置的通识模型作为基础大模型       |

## 技能设定
`技能设定` 部分如下图所示，给专属大模型设置其所能使用的技能，您需要从技能库中选择技能，使得该大模型能支持这些技能。

<img
    src="docs/images/llm-new-others.png"
    alt="new LLM"
/>

## 知识设定
`技能设定` 部分如上图所示，给专属大模型设置其所需要学习的知识，您需要从知识集中选择知识，使得该大模型能学习并运用这些知识。

## 版本描述
每个大模型都是以项目的方式进行版本化管理，可以给每一个版本进行描述。
